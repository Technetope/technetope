# Onboarding Checklist

1. **Read core docs**
   - `acoustics/docs/masterdocs.md`
   - `acoustics/process/coding_rules.md`
2. **Set up environment**
   - PC tools prerequisites (Python/Node/etc.).
   - PlatformIO/Arduino toolchain with PSRAM support.
3. **Verify hardware**
   - Install CH9102 driver.
   - Flash sample firmware and confirm I2S audio.
4. **Run smoke tests**
   - Execute OSC loopback script on PC.
   - Trigger sample playback on one Stick using the draft firmware.
5. **Join communication channels**
   - Slack/Teams channel for acoustics project.
   - Shared calendar for test sessions.

Tick off each item and link related evidence (screenshots, logs) when announcing completion.
