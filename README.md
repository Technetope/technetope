 <h2>Statement</h2>
 <i>L’homme est un roseau pensant(人間は考える葦である)</i> - Blaise Pascal(1623–1662)
 <div class="pt-callout">
    <h5 class="pt-callout-title">Note</h5>
    人間は一本の葦にすぎない。自然のうちでもっとも弱いものである。だが、それは考える葦である。宇宙全体が武装して彼を押しつぶす必要もない。一つの蒸気、一滴の水でも彼を殺すのに十分である。しかし、宇宙が彼を押しつぶしたとしても、人間はなおそれを押しつぶすものよりも高貴である。彼は自分が死ぬことを知り、宇宙が彼にまさることを知っている。宇宙はそれを知らないのだ。
</div>
